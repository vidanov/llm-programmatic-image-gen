Tokens grow linearly (10×). Rendering power grows exponentially. All outputs: editable source code.

The key insight: as you move up the renderer stack, token cost grows linearly but capability grows exponentially. Blender's Python API lets an LLM write 50 lines that produce photorealistic renders. Godot scene files are plain text. The "code as canvas" thesis extends all the way to AAA game engines.
